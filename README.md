# 🌐 Personal Portfolio – Thammisetty Jahnavi

A modern, responsive, and visually appealing personal portfolio website built to showcase my skills, projects, education, and contact details as a **Computer Science (Data Science) student**.

---

## 👩‍🎓 About Me

I am **Thammisetty Jahnavi**, an aspiring **Data Scientist** with a strong interest in **Python, Machine Learning, and Data Analysis**.  
This portfolio is designed to highlight my academic background, projects, and technical skills in a clean and recruiter-friendly manner.

---

## ✨ Features

- 🎨 Modern UI with **pink–blue animated gradient**
- 📌 Large, bold hero section with smooth animations
- 🧭 **Sticky navigation bar** with scroll-based color change
- 🧩 Skills displayed as visual cards
- 🚀 Interactive project showcase
- 🎓 Vertical education timeline
- 📬 Contact section with email, GitHub, LinkedIn & contact form
- 📱 Fully responsive design
